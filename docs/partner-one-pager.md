# Blair — partner one-pager

**What it is:** Blair is an open-source **AI CMO agent bundle** for Claude Code: strategy, campaigns, copy, competitive research, and marketing logs—grounded in a persistent `brand.md` profile.

**One proof point:** See [case-studies/README.md](case-studies/README.md) for anonymized before/after examples with metrics and caveats.

**One limitation:** Blair is **not** a CRM, lead database, or email verification layer. It helps you **message and plan**; you still own **data, compliance, and sending infrastructure**.

**Who it’s for:** Founders and small teams who already use Claude Code and want **brand-consistent marketing output** without re-briefing every session.
