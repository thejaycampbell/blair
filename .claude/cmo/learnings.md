# Blair Learnings Log

This file is written automatically. Every correction, preference, and override gets logged here.
Every specialist reads this before producing any output. It compounds. The longer you use Blair, the more it sounds like you.

---

## How to add a learning

During any session, say:
- "Remember that..."
- "Next time, always..."
- "We never say X, we say Y instead"
- Or just correct an output and say "log that"

Blair writes the entry. You never touch this file manually.
